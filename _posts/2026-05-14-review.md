## 论文平台
按影响力降序排列

| 术语       | 说明                                                                                   |
| -------- | ------------------------------------------------------------------------------------ |
| 影响因子（IF） | 某期刊前两年发表的论文在统计当年的平均被引次数。公式：  <br>`IF = (2024年引用2022–2023年文章总数) / (2022–2023年发表文章总数)` |
| JCR 分区   | 科睿唯安将每个学科的期刊按 IF 降序分为 4 等份：  <br>Q1（前25%）、Q2（25–50%）、Q3（50–75%）、Q4（75–100%）          |
| 中科院分区    | 中国科学院制定的另一种分区标准（1区≈前5%，更严格），需通过 中科院文献情报中心分区表 查询                                      |

被**引用**次数的**来源**文章 **完全不必发表在 Nature 上**，甚至 不需要是英文期刊、SCI 期刊，或正规学术期刊。

| 维度    | 知网（CNKI）                                | 国际顶级期刊（如 Nature、Circulation 等）                |
| ----- | --------------------------------------- | --------------------------------------------- |
| 性质    | 文献数据库平台（类似“图书馆”）                        | 学术期刊（发表原创研究的“出版物”）                            |
| 语言    | 主要为中文                                   | 主要为英文                                         |
| 内容范围  | 收录大量中文学术成果，质量参差不齐（从高质量核心期刊到普通期刊都有）      | 只发表经过严格同行评审的高质量原创研究                           |
| 国际影响力 | 主要在中国大陆使用，国际学术界基本不引用知网文献                | 全球科研界公认，是科研评价、职称晋升、基金申请的重要依据                  |
| 影响因子  | 知网本身没有影响因子；它收录的中文期刊可能有“复合影响因子”，但远低于国际期刊 | 如 Nature IF≈64.8，Circulation IF≈37.8，代表全球顶尖水平 |
| 学术地位  | 是查找中文文献的重要工具，但不是高水平研究成果的标志              | 发表在这些期刊上被视为重大科研成就                             |

了解DOI
```
10.1038 / s41586-026-10346-5
│      │
│      └─── 这是你看到的那串字符（文章唯一ID）
└────────── 注册机构代码（10.1038 = Nature 出版社 Springer Nature 的专属前缀）
```

**DOI**（Digital Object Identifier，数字对象标识符）是**学术文献的永久唯一身份证号**，就像人的身份证号或书的 ISBN。也可以直接通过doi网址访问

```
https://doi.org/10.1038/s41586-026-10346-5
```

作者列表：姓氏全大写（TSABAN），名字缩写为首字母（G.）。
列出前几位作者后，用“et al.”（拉丁语 et alii，意为“等人”）表示还有其他作者（通常在作者超过6人时使用）。
这是医学期刊常见的作者缩写方式。

描述研究内容：使用心电图评估二尖瓣反流患者的舒张功能和预后。

[J]
文献类型标识符：
[J] = Journal article（期刊文章）
其他常见标识：[M] 专著，[C] 会议论文，[D] 学位论文等。
这是中国国家标准 GB/T 7714 推荐使用的文献类型代码，说明该格式可能混合了国际医学格式与中国国标要求。

J Am Coll Cardiol
