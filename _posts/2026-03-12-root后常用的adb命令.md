修改数字密码(PIN码)
```
locksettings set-pin --old 123456 123456
```

禁用bilibili
```
pm disable tv.danmaku.bili
```

禁用抖音
```
pm disable com.ss.android.ugc.aweme
```

锁屏
```
input keyevent 26
```

打开拨号键盘
```
input keyevent 5
```

```
input keyevent 6
```

禁用wifi
```
svc wifi disable
```
