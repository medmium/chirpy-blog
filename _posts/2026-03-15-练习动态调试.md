用IDA进行动态调试的技术已经老掉牙哦, 换FRIDA吧

## frida脚本
---
孵化一个app

```shell
frida -U -f 包名 -l 脚本.js
```

>注意事项: 
- 如果脚本包含**中文**, 请确保文件为**UTF-8**格式
- frida 17.x  `findExportByName`已失效,请使用`Module.getGlobalExportByName("android_dlopen_ext")`

`android_dlopen_ext`位于
```
/system/lib64/libdl.so
```
![](obsidian/attachments/Image_2026-03-18_12-57-31_ncscfwio.zmd.png)

根据[看雪教程](https://bbs.kanxue.com/thread-281584-1.htm), 该函数是加载app自带so的关键函数, 下面开始验证, 自编写一个App调用
```java
static {
        System.loadLibrary("tracerpid");
    }
```

```java
@CallerSensitive  
public static void loadLibrary(String libname) {  
    Runtime.getRuntime().loadLibrary0(Reflection.getCallerClass(), libname);  
}
```
